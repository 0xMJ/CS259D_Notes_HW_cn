## Web 安全

* 从个人到大型组织，Web 应用被广泛使用
* Web 安全影响因素
    * Web 应用市场增长迅速，导致由不同语言、框架和协议开发的Web应用程序也大量增长，这使其难以维护
    * Web 应用容易受到来自世界各地的攻击
    * 一些开发者、管理者和使用者没有经过足够的培训，或者缺乏安全领域所需知识、经验
    * Web 应用提供了一个访问企业资源的途径
